# ADREC

Requirements:

1. `"react-native": "0.76.7"`
2. `Node Version: v20.11.0`

Installation:

Follow the following steps to install the project after cloning it successfully to your system.

1. `npm install`

Running:

1. for running project on android:
   `npm run android`

2. for running project on ios
   `npm run ios`
